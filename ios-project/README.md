# iOS project

`ZhaiwoNativeScan.xcodeproj` is a static framework project based on the DCloud
uni native plugin pattern.

Build on macOS from the repository root:

```bash
HBUILDER_IOS_SDK=/path/to/IOS-SDK/SDK ./scripts/build-ios-framework.sh
```

The plugin module class is `ZhaiwoNativeScanModule`.
